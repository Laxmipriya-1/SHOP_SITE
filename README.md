
# Online Store Homepage with Product Categories🎉

## Project Overview
A dynamic and responsive online store homepage built with React.js, featuring categorized product listings, filter functionality, and smooth UI/UX.

## Features ⚡
* Product categories for men, women, and kids
* Interactive product cards with image, name, price, and "Add to Cart"
* Filters by category and sorting by price/rating
* Responsive design for mobile, tablet, and desktop
* Deployed on Vercel

## Technologies Used
* React.js
* Tailwind CSS
* Git/GitHub
* Vercel

## How to Run the Project
1. Clone the repo: `git clone <https://github.com/Laxmipriya-1/SHOP_SITE>`
2. Install dependencies: `npm install`
3. Start the app: `npm start`
4. Open `http://localhost:5173` in your browser
